Carpeta contenedora de los archivos del proyecto

*ajax 	     => Archivos de peticiones de servicios ajax
*config	     => Archivos de configuracion del sistema php
*files 	     => Archivos cargados por el usuario pdf,png,...
*models      => Clases de los objetos de la BD
*public	     => Hojas de estilos css, Imagenes png, Funciones js, Librerias
*views 	     => Paginas web html y php del sistema
